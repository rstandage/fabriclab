Pre-reqs
0.1) install Proxmox latest
0.2) set static IP on eth0

Process for creating virtual fabric
 1) run apt-update
 2) create directories
 3) download files
 	- vJunos qcow
 	- SSR iso
 	- SRX iso
 	- ubuntu for hosts iso
 	- mist edge iso
 4) sync with git
 5) create networks
 6) set enable_lldp.sh to run on boot
 7) create virtual switches
 8) modify assigned bridges on switches for the required connectivity
 9) adopt switches
10) create switch template
11) create site
12) assign switches to site
13) build fabric
